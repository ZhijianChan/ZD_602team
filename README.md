#### Android Contact Demo 安卓通讯录  
* 2015 weixin-sysu android development program project  
#### 功能介绍  
* 这个软件的主要功能支持 
  * 登录注册 
  * 添加联系人 
  * 关键字搜索 
  * 标签管理 
  * 二维码名片生成与推送 
  
#### 使用方法
* 完成登录注册后，即可进入主界面，以后每次打开都保存登录状态  
* 左上角点击可以打开菜单，可以编辑登录信息、查看标签管理、个人二维码  
* 右上角点击可以查看功能菜单，包括添加联系人、扫二维码、搜索
